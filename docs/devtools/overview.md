# Devtools Overview

> **Coming Soon** — Developer tools are not yet implemented.

Nika Devtools will provide a set of tools for debugging, monitoring, and managing Nika applications.

## Planned Features

| Feature | Description | Status |
|---------|-------------|--------|
| Dashboard | Web-based monitoring dashboard | ⏳ Planned |
| Module inspector | View loaded modules and providers | ⏳ Planned |
| Route inspector | View all registered routes | ⏳ Planned |
| DI visualizer | Visualize dependency graph | ⏳ Planned |
| Log viewer | Real-time log streaming | ⏳ Planned |
| Config viewer | View current configuration | ⏳ Planned |

!!! info "Want to contribute?"
